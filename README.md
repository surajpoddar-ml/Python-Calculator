# Simple Calculator

A basic Python calculator program for performing arithmetic operations.

## 👨‍💻 Author
**Suraj Poddar**

## 📋 Features
- ➕ Addition (A)
- ➖ Subtraction (S)  
- ✖️ Multiplication (M)
- ➗ Division (D)
- 🚪 Exit option (E)
- ⚠️ Error handling for invalid inputs
- 🚫 Prevents division by zero

## 🎯 How to Use
1. Run the program
2. Select an operation (A, S, M, D, or E to exit)
3. Enter two numbers when prompted
4. View your result
5. Continue or exit as desired


## 🔧 Requirements
- Python 3.x or higher

## 🚀 How to Run
```bash
python calculator.py
```

This is a beginner-friendly project demonstrating basic Python concepts like loops, conditionals, and user input handling.
